// Features

What's Inside

🧹

Data Pipeline

Automated data cleaning with Pandas — handles missing values, type conversion, and feature engineering in one function call.

🤖

ML Prediction

Random Forest model trained on sales data to predict revenue. Handles categorical encoding automatically with one-hot encoding.

⚡

REST API

FastAPI backend with auto-generated docs at /docs. Endpoints for prediction, analysis, and business summaries.

🧠

RAG System

Retrieval Augmented Generation using FAISS vector search and Sentence Transformers to answer natural language business questions.

🐳

Docker Ready

Fully containerized application that runs identically on any machine or cloud platform with a single docker run command.

📊

Business Analytics

Automatic aggregation of sales data — total revenue, top products, best regions, and transaction metrics computed on demand.
